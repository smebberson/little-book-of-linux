Users and groups

Switching user

su - {user} -s /bin/sh

This will switch to the specified user, with the shell /bin/sh.

Show all groups

getent group

This will show all groups on the system. The output is broken down as follows:

group:x:24:user1,user2
|     | |  |
1     2 3  4
  1. Group name: the name of the group.
  2. Password: generally password is left blank.
  3. GID: the id of the group.
  4. Group list: the list of users that belong to the group.
